# Lucairn SDKs

Clients for Lucairn (lucairn.eu) — privacy-preserving AI gateway. This monorepo hosts three official SDKs at parity: TypeScript, Python, and Go.

## Status

Pre-1.0. All three SDKs at parity at the observable level; cross-language
byte-equivalence locked via shared Go-assembler-generated fixtures.
